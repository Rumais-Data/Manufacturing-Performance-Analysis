# Manufacturing-Analysis
Manufacturing Analysis using Excel, SQL, Power BI &amp; Tableau

🏭 Manufacturing Performance Analysis Dashboard

* 📌 Project Overview

  This project analyzes manufacturing performance using an interactive dashboard built in Power BI. The dashboard helps monitor production output, rejected   quantities, and machine/employee performance to improve operational efficiency.

* 🛠 Tools Used

  Power BI

  Excel

  SQL

  Tableau


📊 Key Metrics

Total Produced Quantity: 109M

Manufactured Quantity: 72M

Balance Quantity: 28M

Rejected Quantity: 441K

📈 Dashboard Insights

Identifies top performing employees by production output.

Shows top machines contributing to production.

Tracks daily production trends.

Compares manufactured vs rejected quantity to evaluate production quality.

🎛 Filters Available

Department

Production Type

Date

These filters allow users to analyze production data interactively.
